Mini‑GW Web Song Studio

Static HTML you can host on GitHub Pages. Uses a configurable AI endpoint if you provide one, otherwise falls back to a local algorithmic generator and in‑browser synthesis (WAV download).

Output format: WAV (downloadable). If you supply an AI endpoint that returns audio bytes or a WAV URL, the page will use that instead.
After generation completes, a download link will appear here.
Drop this file into a GitHub repo and enable Pages. To use a remote AI, point AI Endpoint to your model server that accepts JSON {prompt} and returns either JSON {notes:[...]} or raw WAV bytes.
